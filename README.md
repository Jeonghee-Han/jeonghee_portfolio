# 백엔드 개발자 한정희 포트폴리오

> 사용자의 안정적인 경험은 탄탄한 백엔드에서 시작된다는 믿음으로, 보이지 않는 곳까지 책임집니다.

<br>

**🚀 배포 주소: [https://jeonghee-portfolio.vercel.app/](https://jeonghee-portfolio.vercel.app/)**

<br>

## 📖 프로젝트 소개
저의 개발 경험과 기술 스택, 성장 과정을 보여주기 위해 제작한 개인 포트폴리오입니다. 프로젝트를 진행하며 겪었던 기술적 고민과 문제 해결 경험을 상세히 담아냈습니다.

<br>

## 🛠️ 기술 스택
* **Language**: `Java`
* **Backend**: `Spring Boot`, `JPA`, `MySQL`, `Redis`
* **Frontend**: `React`
* **Deployment**: `Vercel`, `AWS EC2`
* **Etc**: `Git`, `GitHub`, `Docker`
